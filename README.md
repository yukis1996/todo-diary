# **Todo-Diary**
Todo-Diaryは日々のやること入力しスケジュール管理ができます。  
また1日行ったことを日記にすることによって1日の思考を整理することができます。

1.[機能](#機能)  
2.[デモ画像](#デモ画像)  
3.[開発環境](#開発環境)  
4.[DB設計](#DB設計)  
5.[今後の展開](#今後の展開)

## *機能*
---
* ユーザー登録
* タスクを入力し保存する
* 「今日」に該当するタスクを一覧表示
* タスクの一覧表示及び削除
* 日記の投稿
* 各利用者の日記を閲覧
